What are the best practices associated with using them?


Use functional names, as it supports the structure and makes it easier to make adjustments/conceptualize.

Keep the naming of the Elements as long as they need to be while simoultaneously making them as short as possible.

Only use ID's if you want a style to apply to one element on the web page, and to use a class selector if you want to style many elements.
